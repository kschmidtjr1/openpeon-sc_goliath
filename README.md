# StarCraft Goliath — OpenPeon Sound Pack

*"Goliath online."*

A [peon-ping](https://github.com/PeonPing/peon-ping) sound pack featuring the Terran **Goliath** from StarCraft: Brood War.

## Preview

[Listen to all sounds](https://openpeon.com/preview#kschmidtjr1/openpeon-sc_goliath)

## Sounds

| Category | Clips |
|---|---|
| `session.start` | *"Goliath online."* |
| `task.acknowledge` | *"Acknowledged H.Q."*, *"Nav-comm locked."*, *"Confirmed."*, *"Target designated."* |
| `task.complete` | *"Channel open."*, *"Systems functional."* |
| `task.error` | *[Death]* |
| `input.required` | *"Go ahead, TACCOM."*, *"Comm-link online."* |
| `resource.limit` | *"Running level one diagnostic."* |
| `user.spam` | *"MilSpec E.D.2.0.9. on."*, *"Checklist protocol, initiated."*, *"U.S.D.A. selected."*, *"F.D.I.C. approved."*, *"Checklist completed... S.O.B."* |

## Install

```bash
peon packs install sc_goliath
peon packs use sc_goliath
```
